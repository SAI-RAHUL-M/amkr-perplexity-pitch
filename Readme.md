# AMKR — Packaging is the New Fab
**Perplexity Stock Pitch Competition 2026**

This repository contains a complete financial model, interactive web pitch, and investment thesis for Amkor Technology (AMKR), generated using Perplexity Computer. 

## Project Overview
This project was built for the week-long Perplexity Stock Pitch Competition (March 30 – April 3, 2026), featuring a $17.5k prize pool and judged by the founders of Coatue and Third Point. The thesis positions AMKR not as a commodity assembler, but as the critical physical bottleneck of the AI chip deployment cycle, featuring a comprehensive Sum-of-Parts valuation and Section 232 tariff impact analysis.

## Repository Structure
* `index.html`: The core pitch presentation, designed with a Bloomberg Terminal meets Linear.app aesthetic.
* `style.css`: Premium design system utilizing Space Grotesk and Libre Baskerville.
* `app.js`: Application logic including a dynamic hero canvas, scroll-triggered animations, and interactive Chart.js financial models (DCF, Revenue, EPS beats).
* `model.py`: The quantitative engine backing the pitch, calculating FY2026E segment revenue, ASE re-rating precedents, and government incentive offsets.
