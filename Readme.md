This repository contains a complete financial model, interactive web pitch, and institutional-grade investment thesis for Amkor Technology (AMKR). 

The core thesis challenges the market's classification of AMKR as a commodity OSAT (Outsourced Semiconductor Assembly and Test) and re-prices it as the critical physical bottleneck of the AI chip deployment cycle.

## 🏆 Project Overview
This project was developed for the **Perplexity Stock Pitch Competition**, leveraging Perplexity Computer to execute deep-dive financial research, uncover hidden alpha, and synthesize a comprehensive narrative. 

**Key Pitch Highlights:**
* **Variant Perception:** The market prices AMKR like plumbing (1.65× P/S) while the companies whose equipment it uses trade like infrastructure (10–17×). 
* **The Tariff Tailwind:** Uncovered hidden alpha within Trump's Jan 15, 2026, Section 232 proclamation. AMKR's Arizona facility is explicitly exempt from the 25% semiconductor import tariff, while Chinese competitors (JCET) face severe cost disadvantages.
* **ASE Historical Precedent:** Mapped AMKR's current trajectory against ASE Technology's 2018–2025 re-rating (0.62× → 1.74×), showing an exact parallel setup.
* **Smart Money Signals:** Tracked recent congressional buying activity (e.g., Armed Services Committee) signaling recognition of AMKR as critical national security infrastructure.

## 🛠️ Repository Structure
The pitch is delivered as a fully interactive, single-page application with a dynamic financial engine.

* `index.html`: The core presentation layer, featuring a Bloomberg Terminal inspired UI, structured thesis arguments, and interactive data visualization.
* `style.css`: A premium design system utilizing Space Grotesk and Libre Baskerville, complete with a dark/light mode toggle and custom animations.
* `app.js`: Application logic powering the dynamic hero canvas, scroll-triggered fade-ups, and interactive `Chart.js` financial models.
* `model.py`: The quantitative Python engine backing the pitch. It calculates FY2026E segment revenue, Sum-of-Parts (SOTP) valuation, ASE re-rating precedents, and government incentive ($2.85B) capex offsets.
